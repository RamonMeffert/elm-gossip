# Tools for Gossip

BSc Project

## Installing

If you want to build this project yourself, you'll need [`yarn`][1] or [`npm`][1].

To install:

```sh
yarn install
# or
npm install
```

## Building and running

Build a release:

```sh
yarn build
# or
npm build
```

Run a live-reload server in development mode:

```sh
yarn start
# or
npm start
```
